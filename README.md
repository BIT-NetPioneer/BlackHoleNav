BlackHoleNav
------
这是个基于CodeIgniter框架，php编写的导航站

Install
------
 * 自备apache+php+mysql环境
 * 导入**sql**文件夹下的sql文件，其中**nav.sql**是数据库结构，**demo_data.sql**是测试用数据
 * 修改sql连接参数**application\config**下的**database.php**
